# Teste Técnico de Desenvolvimento Mobile Remopt

## Como Rodar o projeto no seu computador

## Pré-requisitos

Antes de rodar o projeto, é necessário que seu computador esteja configurado com os seguintes itens:

- **Node.js** (versão LTS recomendada)
- **Yarn** ou **npm**
- **Java Development Kit (JDK)** 11 ou superior
- **Android Studio** instalado e configurado
  - Android SDK
  - Android SDK Platform
  - Android Emulator ou dispositivo físico com depuração USB ativada
- Variáveis de ambiente configuradas:
  - `ANDROID_HOME`
  - Inclusão do `platform-tools` no `PATH`
- **Git** para clonar o repositório



0. Clone o Repositorio no seu computador

1. Instale as dependências do projeto:
```shell
yarn install
```

2. Rode o comando para ativar o servidor localmente:
```shell
yarn react-native run-android
```
